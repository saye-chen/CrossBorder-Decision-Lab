# LIFD 跨域集成协议

LIFD 拥有路线、仓网、库存、补货、ATP/CTP、履约、逆向和海外库存处置主权。CIDM 提交资本边界，PLCO提交页面承诺需求，AAMO提交流量情景，CIG提交退货/VOC汇总，CIM提交外部供应履约代理；LIFD只接收本域所需字段。

交接卡含 `object_id`、SKU/批次、国家/仓/渠道、版本、`as_of_time`、数量单位、证据/计算 ID、允许用途、禁止用途、置信区间和状态。LIFD不得直接改采购合同、价格、广告、页面或客户补偿；只输出 `proposed/blocked/inconclusive`。

部分失败保留可对账库存和已验证节点，相关承诺降级。冲突按安全/召回红线、货权、数量守恒、主权、时效裁决。回写包含责任、成功、停止、回滚、退出与实际结果。
