# CIDM 专业报告交付

Decision Card 用于单对象快速判断；Decision Memo 用于进入、组合、复盘和生命周期动作；Investment Diligence 用于投委会、多国家平台或不可逆资本决策。三层只改变展示长度，不改变研究门。

任何层级必须含对象/版本、结论、证据与反对证据、门槛、利润或资本约束、动作、成功、停止、回滚和缺失影响。Memo 增加替代方案、不行动基线、敏感性和责任；Diligence 再增加完整 Ledger、三情景、跨域结果与退出价值。

报告必须标注 L1 Structure、L2 Contract、L3 Expert、L4 Production 状态；真实回放未通过不得称 production ready。压缩时可折叠附录，不得删除红线、反事实或退出条件。
