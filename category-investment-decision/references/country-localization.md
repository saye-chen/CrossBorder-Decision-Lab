# 国家本地化决策

## 路由原则

正式结论必须落到具体销售国家、平台和进口/履约模式。“欧洲”和“东南亚”只是区域路由，不是统一税务或合规市场；用户只给区域时，先选择 2-4 个候选国家比较，再推荐进入顺序。

本文件覆盖常见区域。遇到未列明国家、区域或本地规则不确定时，读取 `country-routing-universal.md`，按十项国家检查重新核验。

消费者偏好、价格、税率、关税、认证、标签、平台费、进口门槛和物流都可能变化，动态费用、平台政策、法规红线、税务、物流和品类相关要求必须在报告时核验当前来源。稳定框架可以作为分析结构使用，但不得替代会影响利润、合规或进入条件的当前证据。不要把本文件当法规或税率数据库。

单国家专业报告不做无关国家横向扩展，必须把深度集中到目标国家内部：本地需求、渠道准入、税费/费用、品类法规、进口履约、语言内容、售后体验和进入前复核材料。若某项会改变进入结论、利润或合规红线，必须局部深挖；若只是一般背景，压缩展示。

## 八项本地化检查

| 模块 | 必查内容 |
|---|---|
| 消费需求 | 本地使用场景、审美、尺寸、气候、宗教/文化、痛点与支付意愿 |
| 平台渠道 | 主流平台、搜索/内容/活动流量、线下替代、账号准入与本地履约 |
| 价格税务 | 含税/未税展示、VAT/GST/销售税、平台代征、申报和汇率敏感性 |
| 产品法规 | 监管分类、认证/测试、责任主体、标签、说明书、营销宣称和召回责任 |
| 进口海关 | HS 归类、关税、进口主体、原产地、许可证、禁限运与清关资料 |
| 物流售后 | 头程、仓配、偏远区、COD、退货地址、逆向物流、破损和时效预期 |
| 语言内容 | 法定语言、计量单位、客服、搜索词、素材语境与本地达人表达 |
| 经营适配 | 竞争壁垒、现金周期、首批库存、团队能力、本地服务商与退出成本 |

所有成本统一换算到同一币种和同一含税/未税口径，再执行原利润模型。汇率至少做基准和压力场景。

## 区域路由

### 美国

- 明确联邦要求、州级差异和平台责任边界；优先核验相关监管机构、CBP、税务机关和平台官方规则。
- 按品类路由到 FDA、CPSC、FCC、FTC、EPA 等适用机构，不因“在 Amazon 可售”推断法规合规。
- 检查英制/公制标示、英文标签、产品责任、州级化学品/回收要求、销售税处理和本地退货体验。
- 分别测算 Amazon FBA、第三方仓和自发货，不把平台代征税重复计入成本。

### 欧盟与英国

- 先确定具体成员国；欧盟统一规则与成员国执行、语言、EPR 和税务要求需同时核验。
- 按品类检查 GPSR、CE 适用性、REACH、WEEE/电池、包装/EPR、责任人、标签语言、VAT/IOSS 和海关要求；只核验真正适用的制度，不堆认证名称。
- 英国单独评估，不把欧盟 VAT、责任人、标志或市场准入规则直接套用到英国。
- 将多语言包装、责任人、EPR、退货地址和较长合规周期计入启动预算。

### 日本

- 核验进口销售主体、JCT/关税、日文标签和品类主管部门；按产品判断 PSE、PSC、无线、食品、化妆品/医药部外品等要求。
- 本地消费者通常对尺寸、说明、包装完整性和客服预期较高；必须用本地评论与竞品页面验证，不以刻板印象评分。
- 比较 Amazon Japan、Rakuten、Yahoo! Shopping 等渠道的准入、费用、流量与履约。
- 将日文化、包装适配、检测、进口商和逆向物流计入成本与交期。

### 东南亚

- 必须指定国家：新加坡、马来西亚、泰国、菲律宾、越南、印度尼西亚等在税务、语言、认证、进口和支付方式上不同。
- 核验 Shopee、Lazada、TikTok Shop 等平台在目标站点的费率、活动、COD、退货和本地仓规则。
- 按国家与品类检查清真、化妆品、食品、电子、电信、消费者保护和标签要求；“东南亚可卖”不是合规结论。
- 将 COD 取消、岛屿/偏远区、跨境时效、本地币汇率和低客单价压力纳入利润压力场景。

## 国家比较方法

对每个国家仍输出原七维基础分；不要新增一套替代总分。另加不计入总分的“本地化准备度”：

| 状态 | 判断 |
|---|---|
| 已就绪 | 责任主体、合规路径、税务、标签和履约方案均已验证 |
| 有条件 | 路径明确但报价、注册、测试或服务商尚未落地 |
| 未就绪 | 关键责任主体、许可、标签或清关路径不明确 |

比较国家时统一产品规格、到岸成本日期、汇率日期和销售阶段。最终进入顺序同时考虑总分、本地化准备度、启动预算、上市时间和退出成本。

## 输出协议

1. 国家与销售模式边界：平台、进口主体、履约、币种、含税口径和日期。
2. 本地消费者与需求：使用场景、价格带、关键词和内容语境。
3. 渠道结构：平台流量、竞争、准入和履约。
4. 法规/税务/进口清单：适用、可能适用、不适用/未发现，并附官方来源。
5. 本地单位经济：汇率、税费、关税、仓配、退货和三场景利润。
6. 本地化改造：规格、包装、语言、说明、客服、支付和内容。
7. 五道门槛、七维评分与本地化准备度。
8. 进入顺序：先进入国家、候补国家、暂缓国家及触发条件。

单国家报告将第 8 项改为“进入路径”：首发渠道、SKU/价格、合规/税务/物流补数、测试预算、Go/Stop 和人工复核项。

高风险法律或税务问题必须建议当地专业人士复核；不要把一般信息表述为法律、税务或海关意见。
