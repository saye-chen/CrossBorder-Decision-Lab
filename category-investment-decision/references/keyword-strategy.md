# 关键词策略

## 定位

关键词不是独立词库任务，而是投决证据和上市打法的一部分。它支持市场需求、竞争可进入性、内容传播、利润空间和放量后的广告效率判断。

使用场景：Amazon 长尾机会、ASIN 关键词反查、广告词包诊断、Listing 埋词、内容选题、否定词清单、竞品流量缺口。

## 八维关键词分类

| 维度 | 定义 | 策略含义 |
|---|---|---|
| 核心词 | 品类主干词和大词 | 判断需求规模，长期经营，不轻易用低预算硬抢 |
| 功能词 | 描述功能、效果、解决方案的词 | 广泛拓流，验证卖点是否被搜索 |
| 场景词 | 使用场景、人群、节日、空间、任务 | 拆广告组和内容角度，寻找细分切入 |
| 属性词 | 尺寸、颜色、规格、数量、兼容性等限定词 | 长尾精准匹配，转化率通常更高 |
| 材质/成分词 | 材质、成分、工艺、认证相关词 | 差异化和风险复核重点 |
| 品牌词 | 自有品牌、竞品品牌、泛品牌词 | 自有防守、竞品策略判断；竞品投放需注意平台规则 |
| 否定词 | 无关流量、误匹配、敏感或违规词 | 控制 ACOS/CPA，减少低质曝光 |
| 机会词 | 竞品弱覆盖、内容强、转化强或新趋势词 | 优先测试，连接切入楔子 |

关键词分类必须保留“为什么归类”的简短依据，避免只按词面机械分类。

## 生命周期打法

| 阶段 | 关键词重心 | 打法 |
|---|---|---|
| 新品期 | 长尾精准词、属性词、场景词 | 低竞争高转化，先拿成交和评论，不追求大词覆盖 |
| 成长期 | 场景词、功能词、机会词 | 扩量测试，按转化和净贡献筛词 |
| 成熟期 | 核心词、自有品牌词、高贡献词 | 防守排名，优化自然单占比和广告依赖 |
| 防御期 | 竞品品牌词、机会词、替代场景词 | 抢位、找新增量、修复流量下滑 |

## 投决连接

| 投决维度 | 关键词证据怎么用 |
|---|---|
| 市场需求 | 关键词簇数量、趋势、搜索意图、购买频次和场景宽度 |
| 竞争可进入性 | 首页评论结构、广告密度、长尾词新品突破、竞品覆盖缺口 |
| 利润空间 | CPC/CPA 代理信号、盈亏平衡 ACOS、低价词依赖程度 |
| 内容传播 | 场景词、问题词、结果词是否能转成 3 秒钩子和 UGC 模板 |
| 机会窗口 | 新兴词、季节词、平台扶持词、竞品未覆盖词 |

关键词不能单独推出“市场大”；必须和商品页、排名、评论增长、平台趋势或用户数据交叉验证。

## 输出模板

关键词分析默认输出：

1. 输入范围：ASIN/品类/站点/时间窗口/数据来源。
2. 关键词总览：词量、核心词簇、意图结构、明显噪音。
3. 八维分类表：每类代表词、商业含义、动作。
4. 机会词清单：词、证据、竞品覆盖、建议动作、验证指标。
5. 否定词清单：词、否定原因、适用范围。
6. Listing/广告/内容三端应用：埋词、广告组、素材角度。
7. 对评分和门槛的影响：哪些维度上调、下调或保持不变。

结构化导出时，可额外生成 `keywords.csv`、`negative_words.txt` 和分类摘要；用户未要求文件时直接在对话中交付。
