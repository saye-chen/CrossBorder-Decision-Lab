# CIDM 对接

只有用户提供品类投资报告或明确要求回写时启用。VLB 独立运行时不读取或修改 CIDM 历史报告。

## 输入

读取 CIDM 模型版本、证据截止日、内容传播评分、利润评分、风险评分、切入楔子、最弱假设和五道门槛状态。

## 回写映射

- 产品-视频匹配与可复制性 → 内容传播；
- 单位经济和达人/素材成本 → 利润空间；
- 产品与内容合规 → 风险可控性；
- 新视频证据 → 切入楔子、验证计划或机会窗口。

## 输出协议

输出结构化“建议调整”，包含原 CIDM 版本、原分、建议分、新增证据/假设编号、调整理由、置信度、是否触发五道门槛重判和下一步验证。

不静默覆盖历史报告，不因 CIDM 原结论乐观而弱化 VLB 发现的内容、利润或合规否决项。未经用户确认不得修改旧评分或文件。
