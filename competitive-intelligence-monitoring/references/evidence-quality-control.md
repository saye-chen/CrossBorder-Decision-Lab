# 证据与质量控制

## 证据标签

| 标签 | 含义 |
|---|---|
| O | 页面、素材或商品的直接观测 |
| M | 实测、平台或授权工具数据 |
| U | 用户提供的业务数据 |
| B | 有来源和日期的外部基准 |
| E | 销量、份额、广告等代理估算 |
| I | 由上述证据导出的推断/归因假设 |

每条数据保留平台、市场、时间、URL/文件位置、采集条件和是否为代理指标。估算附方法、区间、假设和置信度。抓取失败、链接失效和缺失值显式标记。

## 发布前检查

- 事实、估算、推断和建议可分辨。
- 归因有置信度、支持和反驳信号。
- 时效在最近 2 个监控周期内；更旧标记历史。
- 趋势不基于单点，价格对比优先覆盖 ≥3 个时点。
- 公开评论的比例只称“样本内”；样本 <30 时不声称稳定频率。
- 每条动作包含对象、时间窗口、风险、验证与停止条件。
