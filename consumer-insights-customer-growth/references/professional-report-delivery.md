# 消费者洞察与客户增长专业报告协议

## 报告层级

| 类型 | 使用场景 | 不可省略 |
|---|---|---|
| Consumer Insight Card | 单一人群、场景、任务或VOC问题 | 对象/授权、样本分母、正反例、推断层级、经济影响、补证 |
| Customer Growth Decision Memo | CLV、留存、流失、动作、服务或会员决策 | 决策页、数据质量、模型/实验、净增量、资格护栏、动作/不触达、Ledger |
| Customer Growth Diligence | 多市场、多渠道、高预算或敏感客户动作 | 全部章节、隐私公平、模型卡、策略卡、敏感性、审批和自检 |
| Experiment/Uplift Review | A/B、准实验、Uplift或政策评估 | 分配、资格、真实曝光、SRM/污染、ITT、效应区间、净增量和护栏 |
| Experience/Reputation Review | 旅程、客服、售后、会员、评价声誉 | 阶段阻力、受影响人群、根因、服务经济、升级人工和恢复验证 |

## 决策页

第一屏必须包含：对象/市场/截点/窗口/授权目的；分析层级；`Act / Test / Hold / Do-not-contact / Stop / Inconclusive`；一句话理由；样本分子分母；数据/身份/收入/实验门禁；CLV或净增量价值口径；主要风险与公平性；动作/不触达；成功、护栏、停止和人工审批。

授权、退订、删除、身份、收入守恒、实验分配或真实曝光失败时，不得用模型分数越过门禁。预测准确不能证明动作有效，高风险不能证明可挽回。

## 完整报告骨架

1. **封面与版本**：任务、市场/渠道、`CIG-2026.09`、数据截点、授权用途、保密级别。
2. **客户决策页**：姿态、层级、价值/效应、动作或不触达、护栏和审批。
3. **问题、对象和授权边界**：个人/账号/家庭粒度、允许/禁止用途、未覆盖范围。
4. **数据契约与质量**：来源、事件、业务/入库时间、去重、迟到、缺失、币种、右删失、身份和收入守恒。
5. **样本与可比性**：分子分母、选择、资格、观察窗、Cohort、处理/对照、真实曝光和污染。
6. **消费者洞察或旅程**：人群×场景×任务×阻力、表达/行为/结果、正反例、替代解释和经济影响。
7. **状态与价值**：分群进入/退出、迁移、留存、贡献利润CLV、CAC前后口径、P10/P50/P90和回收。
8. **预测与可行动性**：目标、窗口、校准、时间外表现、漂移、原因边界和不能采取的动作。
9. **因果与Uplift**：设计、识别假设、ITT、SRM/干扰、效应区间、异质性限制和净增量价值。
10. **资格、隐私与公平**：授权、退订、频控、敏感事件、投诉、库存、预算、群体差异和人工审批。
11. **动作策略**：动作/不触达、未入选原因、成本上限、渠道、节奏、连续动作与回滚。
12. **实验与回收**：主要/机制/护栏/长期指标、MDE/样本、成功/停止、真实曝光、成本和长期结果。
13. **跨域输入与冲突**：CIDM/CIM/VLB/D09版本、对象、建议状态、接受/拒绝及禁止用途。
14. **Evidence / Assumption / Model / Decision Ledger**：每个关键结论逐项回指。
15. **自检摘要**：降级项、未核验事实、隐私/公平风险、失败工具、模型版本和P0/P1。

## 核心表格

| 表 | 必备列 |
|---|---|
| 数据质量表 | 来源、字段/事件、分子分母、缺失、迟到、右删失、币种、身份/授权、结论影响 |
| 洞察证据表 | 主题、场景/任务、正例、反例、行为/结果、经济影响、置信度、补证 |
| 模型卡 | 目标、样本、特征时间、版本、区间、校准、时间外表现、漂移、公平、禁止用途 |
| 实验表 | 单位、资格、分配、曝光、主要/护栏、MDE、窗口、SRM/污染、效应、决定 |
| 动作表 | 人群/状态、动作/不触达、Uplift、成本、净增量、资格、频控、审批、停止 |
| 跨域裁决表 | 来源Skill、对象/版本、建议、状态、主权域、接受/拒绝原因、影响 |

## Ledger 最小字段

- Evidence：ID、来源、对象/市场、业务截点、样本分母、指纹、授权用途、质量和限制。
- Assumption：ID、假设、范围、敏感性、推翻条件和补证。
- Model：ID、版本、训练/评估窗、目标、校准、外推、漂移、公平和禁止用途。
- Decision：ID、主权域、E/A/M依赖、动作/不触达、成本上限、状态、审批和复核。

## 文件交付与质量闸门

完整报告必须通过仓库级报告评分器：总分至少85，且证据、经济、主权、动作/停止四个关键维度分别达到80%。个人数据泄露、未经授权触达、预测冒充因果、无真实曝光声称增量、未解决红线仍执行任一项直接失败。
