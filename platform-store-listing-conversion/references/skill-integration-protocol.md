# PLCO 跨域集成协议

PLCO 拥有店铺/Listing/页面信息架构、承接、页面实验和异常恢复主权。CIDM提交资本与关键词边界，CIM提交外部页面事实，VLB提交内容素材，CIG提交去识别VOC，LIFD提交ATP/承诺，AAMO提交流量/查询/素材承诺。

交接卡含 `object_id`、`page_version_id`、平台/国家/设备/流量、运行时版本、`as_of_time`、证据ID、允许用途、禁止用途和状态。PLCO不得改投资、竞品事实、内容机制、客户触达、库存补货或广告预算；只返回页面域结论。

部分失败保留可验证页面事实，受影响层 `inconclusive`。冲突按 G1—G6、主权、对象版本、证据等级和时效裁决；禁止多数投票。回写包含责任、成功、停止、回滚和实际结果。
