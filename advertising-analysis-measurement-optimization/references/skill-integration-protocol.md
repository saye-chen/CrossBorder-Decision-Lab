# D09 跨域集成协议

D09 拥有广告账户结构、预算、出价、优化目标、测量、增量及 Scale/Stop 主权。CIDM 提交资本与利润上限，VLB 提交素材机制候选，CIG 提交去识别人群/增量证据，CIM 提交竞品广告代理，D07 提交库存与履约上限，D08 提交页面承接状态。

结果卡含 `object_id`、广告对象层级、平台/国家、运行时版本、`as_of_time`、证据/计算 ID、成熟度、允许用途、禁止用途和状态。D09 不得改价格、补货、页面、内容、客户触达或投资结论；向这些域只发 `proposed/blocked/inconclusive`。

部分失败时保留已核验的广告事实，受影响的归因/利润/增量层降级。冲突按不可补偿红线、主权、对象版本、证据等级和时效裁决，不多数投票。任何回写都含责任人、成功、停止、回滚和结果回填。
